                        IFR步兵程序说明文档
一、工程名称：IFR_Hero_2018

二、文件夹：
　　　KEIL：编译环境IDE
　　　STM32MCU:stm32相关支持文件
　　　USER:主函数
　　　ROBOT:步兵控制的相关文件
　　　   BSP:板级初始化文件
　　　   APP：机器人控制应用文件
　　　   ALGO:机器人算法文件
　　　   MODULE:机器人外接模块
　　　      MPU6050:mpu6050相关文件
　　　
三、键位操纵：
　　　模式一：遥控器左拨杆为上（remoteData.Switch_Left==1）
云台不工作，底盘随着摇杆实现前后左右转向运动。
　　　模式二：遥控器左拨杆为中（remoteData.Switch_Left==3）
云台处于中值位置，底盘随着摇杆前后左右转向运动，转向运动利用底盘跟随效果；右拨杆为中时，激光打开，摩擦轮转动，右拨杆为下时，激光打开，摩擦轮转动，且拨盘转动，发射子弹。
　　　模式三：遥控器左拨杆为下（remoteData.Switch_Left==2）
此模式为鼠标键盘控制，WSAD分别控制前后左右运动，鼠标左右移实现底盘跟随效果，使底盘实现转向运动；Shift键开启摩擦轮，鼠标左键实现打击效果（点按单发，长按连发）；鼠标右键进入小蛮腰状态（云台不受底盘影响），松开右键则取消该模式。
